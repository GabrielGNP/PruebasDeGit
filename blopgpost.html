Curso Profesional de Git y GitHub

En este blog se van a describir los distintos conceptos aprendidos sobre Git y GitHub en el curso Profesional de Git y GitHub de Platzi dado por Freddy Vega





$ git --version : permite mostrar en la consola la version de git instalada en la maquina



$ git : escribe en pantalla informacion breve sobre cada uno de los comandos que se pueden utilizar con git



$ pwd : escribe en la consola la direccion actual del directorio. La carpeta actual donde se est치 direccionando. Escribe la direccion completa



$ cd : (change directory) permite cambiar el de carpeta referenciada, cambiar de directorio de trabajo. Si se escribe $ cd .. entonces volver치 al directorio padre



$ ls: Escribe en pantalla los archivos y carpetas dentro del directorio referenciado.
       Si se utiliza el argumento -l entonces mostrar치 los archivos y carpetas listados uno debajo de otro. $ ls -l
       Si se utiliza el argumento -a entonces tambien mostrar치 los archivos ocultos dentro del directorio. $ ls -a
       Pueden combinarse los arbumentos -a y -l para mostrar todos los elementos, incluidos los ocultos, listandolos uno debajo de otro. -al o -la, el orden es indiferente.
      $ ls -al



$ clear : Permite limpiar la consola de todas las instrucciones dadas. Tambien puede hacerse CTRL + L para limpiar la pantalla sin necesidad de ejecutar ninguna instruccion.
